🌤️ Weatherly is a dynamic weather app built using Python, FastAPI, SQLAlchemy, and OpenWeather API that allows users to:

  • Check current weather and detailed conditions for any location 🌎
  • View a 5-day weather forecast 🗓️
  • Save, update, and manage their weather searches with full CRUD functionality 💾
  • Use geolocation to get instant weather data for their current location 📍

🌟 This project is part of the PM Accelerator AI/ML Internship, providing real-world AI experience and learning industry best practices.
